Ubuntu マシンに挿したグラボの負荷状況をターミナルで表示する。

git clone https://github.com/Syllo/nvtop.git
sudo apt install build-essential cmake libncurses-dev
mkdir -p nvtop/build
cd nvtop/build
Make ..
sudo make install

このサイトでは icons8 の素材を使用しています。
